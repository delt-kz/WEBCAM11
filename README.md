# Webcam11 Backend API

Backend API for webcam project with CRUD operations.

## Endpoints

- `GET /` - API information
- `GET /api/items` - Get all items
- `GET /api/items/:id` - Get item by ID
- `POST /api/items` - Create new item
- `PUT /api/items/:id` - Update item
- `DELETE /api/items/:id` - Delete item

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the server:
```bash
npm start
```

For development:
```bash
npm run dev
```
